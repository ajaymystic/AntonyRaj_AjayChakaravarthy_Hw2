A Stateful Counter Component

A lightweight JS OOP class that encapsulates state and DOM updates-extensible via inheritance.

Overview

What it does

Counter wraps state and DOM updates into one reusable class. Each instance owns its own container, count, and rendering making multiple independent counters trivial to add.

Usage

How to use it

1. Add a container in HTML

<div id="my-counter"></div>

2. Import and instantiate Counter

import { Counter } from './counter.js';
                        const myCounter = new Counter('#my-counter', 5);

3. Or use StepCounter

import { StepCounter } from './StepCounter.js';
                        const step = new StepCounter('#my-counter', 0, 5);
